# KelMar Tool Tracker

A simple tool sign-out and van stock-take app for KelMar Projects.

- **Sign tools out/in** by scanning a QR sticker — records who took it, with a timestamped photo.
- **QR labels** printable for every tool.
- **Van stock-take** with minimum levels that auto-build an order list.
- **Crew requests** ("Dalton needs 10 boxes of screws") come back to Mitch by name.

Live app is served from GitHub Pages (`/docs/index.html`). Shared data, login, and photo
storage run on Google Firebase (Firestore + Anonymous Auth).

Built for Mitch @ KelMar Projects.
